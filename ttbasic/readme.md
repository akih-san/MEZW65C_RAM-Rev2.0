# 豊四季タイニーBASIC
豊四季タイニーBASICは、「タイニーBASICをCで書く」の著者である鈴木哲哉さんがC言語で作成された、TinyBASICです。<br>
以前、SBCV20/8088ボードに移植したものを、MEZW65C_RAMで動作するようにしました。<br>
エミュレーションモードと、ネイティブモードで動作するバイナリを用意しました。<br>
